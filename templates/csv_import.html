Auto-detected statuses: Tools with "missing", "broken", "damaged", "out of service", "not in spec", or "rejected" in the Notes column are automatically set to the appropriate status.

Duplicate serial numbers are matched to existing tools — notes and certificate info are merged, but existing tools are not overwritten. Re-importing the same file is safe.
